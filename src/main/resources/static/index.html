Fall of the Gods Data

Fall of the Gods data is an API REST service that currently serves as a query for characters, stories, items, worlds and places of the story, as well as the official video game,
and as a query of the achievements and users registered in every video game inside Fall of the Gods' universe.

2.3.0

Errors now have a specific and custom format to be shown in the response body.

2.2.0

Pagination endpoints now are not the only way to list data, free listing endpoints are back for a complete query of all the data an endpoint can show.

2.1.0

Now, all listing endpoints are paginated by default (and at the same time, that pagination can be modified in the same endpoint via request params).

2.0.0

BREAKING VERSION.
New endpoints to access Achievements from Fall of the Gods games (Leaves and Shadows). At the same time, new endpoints to access selected user information and their unlocked achievements.

The main changes that might imply the breaking with previous versions are the refactoring of every endpoint and the new request/response management
now. The old API KEY functionality has been completely deleted.

1.0.3

Automatic code documentation generation and important code refactoring.

1.0.2

API KEY system implementation in order to ensure more security when accessing to data.

1.0.0

It is possible to access all the basic information of the characters, items, worlds, places and stories of both the main lore
and the one selected for the official video game of Fall of the Gods through the provided endpoints.

Public documentation is provided.
